### Image Specifications
- **Format**: SVG (scalable vector graphics)
- **Size**: 400x600px viewBox (2:3 aspect ratio, matching card dimensions)
- **Style**: Kawaii/anime-themed designs with consistent art style
- **Features**: Each card has unique colors, characters, and decorative elements
- **Accessibility**: All designs are visually distinct for easy matching

### Notes
- SVG files are scalable and will look crisp at any size
- Images are displayed using Next.js Image component with `object-fit: cover`
- Cards use a 3D flip animation, so designs are clear and recognizable
- All designs follow a consistent kawaii aesthetic with pink/purple color scheme
